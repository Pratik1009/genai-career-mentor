
# 🚀 GenAI Career Mentor

---

## 📌 Project Overview
GenAI Career Mentor analyzes resumes using Generative AI and provides actionable insights such as ATS score, skill gaps, and interview preparation guidance.

---

## 🎯 Features

- ATS Resume Score
- Job Description Match Percentage
- Missing Skills Detection
- Recommended Skills
- Salary Range Estimation
- LinkedIn Profile Tips
- Course Recommendations
- AI Interview Question Generator
- Mock Interview Chatbot
- Downloadable PDF Career Report

## ⚙️ Installation

Clone repository:

```
git clone https://github.com/yourusername/genai-career-mentor.git
```

Install dependencies:

```
pip install -r requirements.txt
```

Run application:

```
streamlit run app.py
```

---

## 🌐 Deployment

This project can be deployed using **Streamlit Cloud**.

Steps:

1. Push project to GitHub
2. Go to https://share.streamlit.io
3. Connect GitHub repository
4. Select `app.py`
5. Add your `GOOGLE_API_KEY` in Secrets

---

## 🔐 Environment Variable

Create `.env` file:

```
GOOGLE_API_KEY=your_gemini_api_key
```
